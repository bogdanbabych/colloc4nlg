Dear Dr. Babych,

Please find attached a zip file containing the keywords and key phrases extracted from 12 sample creative briefs. The zip file contains 8 files. Here is a brief description of the content of each file:

	•	text.csv: 12 text snippets inside double quotation marks, each corresponding to a creative brief
	•	noun_phrases.csv: a list of extracted noun phrases, each line corresponds to a creative brief 
	•	named_entities.csv: a list of extracted named entities, each line corresponds to a creative brief 
	•	kw_textrank.csv: a list of extracted keywords using TextRank algorithm (Mihalcea, R., & Tarau, P. (2004, July). TextRank: Bringing order into texts. Association for Computational Linguistics.) , each line corresponds to a creative brief
	•	kw_singlerank.csv: a list of extracted key phrases using SingleRank algorithm (Hasan, K. S., & Ng, V. (2010, August). Conundrums in unsupervised keyphrase extraction: making sense of the state-of-the-art. In Proceedings of the 23rd International Conference on Computational Linguistics: Posters (pp. 365-373). Association for Computational Linguistics.) , each line corresponds to a creative brief
	•	kw_sgrank.csv: a list of extracted key phrases using SGRank algorithm (Danesh, Sumner, and Martin. “SGRank: Combining Statistical and Graphical Methods to Improve the State of the Art in Unsupervised Keyphrase Extraction”. Lexical and Computational Semantics (* SEM 2015) (2015): 117.) , each line corresponds to a creative brief
	•	kw_semantic_net.csv: a list of extracted key words by ranking nodes in a semantic network of terms , each line corresponds to a creative brief
	•	kw_azure.csv: a list of extracted key phrases by using Azure Text Analytics Service , each line corresponds to a creative brief

Please let me know if you have any questions. Thanks.
